# System Overview
**SSH Sentinel** is a telemetry-driven honeypot system that captures and analyzes SSH attack activity.
Deployed on Azure to study real-world intrusion behavior and attacker profiling.

---

## Core Features
- Collected **200K+ SSH attack logs** over several weeks
- Parsed and normalized data into structured telemetry
- Visualized attack sources and payloads using **Power BI**
- Extracted **geo-location and frequency distributions** for threat intelligence

---

## Runtime Stack
- Azure VM honeypot environment
- `PowerShell` `Log Analytics` `Power BI`
- Python (ETL and GeoIP processing)
