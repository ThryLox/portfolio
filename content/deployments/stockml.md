# Overview

**StockML** predicts stock trends using multiple ML models. Combines financial data science with interactive web analytics.

---

## Features

- **Five ML algorithms:** Linear Regression, Decision Trees, Random Forest, SVM, XGBoost
- Live ticker data with model performance visualization
- Interactive UI for model comparison and backtesting
- REST API for real-time inference

---

## Tech Stack

`Python` `Flask` `Scikit-Learn` `Pandas` `Plotly`
