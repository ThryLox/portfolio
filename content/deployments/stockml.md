# System Overview
**StockML** is a web-based machine learning application that predicts stock trends using multiple trained models.
Combines financial data science and interactive analytics in a modular deployment.

---

## Core Features
- Implements **five ML algorithms** (Linear Regression, Decision Trees, Random Forest, SVM, XGBoost)
- Fetches **live ticker data** and visualizes model performance
- Interactive UI for model comparison and back-testing results
- REST API integration for real-time inference requests

---

## Runtime Stack
- `Python` `Flask` `Scikit-Learn` `Pandas` `Plotly`
- Frontend: HTML/JS (Flask templates)
- Deployed on Vercel with server-hosted ML backend
