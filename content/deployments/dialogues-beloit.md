# Overview

**Dialogues** is a modern discussion platform built for thoughtful community conversations. Reddit-style threading meets premium UI design.

---

## Features

- **Real-time Discussions** — Start conversations with location tagging and topics
- **Threaded Comments** — Nested reply system for deep discussions
- **Community Voting** — Upvote/downvote to surface quality content
- **Moderation Tools** — Authors can close discussions

---

## Architecture

- **Frontend:** Next.js 14 with App Router
- **Backend:** Supabase (Auth + Realtime + Storage)
- **Database:** PostgreSQL with row-level security
- **Styling:** Tailwind CSS with glassmorphism effects

---

## Design

Dark mode interface with particle physics backgrounds. Built for mobile, tablet, and desktop.

---

## Tech Stack

`Next.js 14` `TypeScript` `Supabase` `PostgreSQL` `Tailwind CSS`
