# Overview

**Aegis Sentinel** is an autonomous cloud security auditor designed with strict zero-trust permission boundaries specifically for LLM agents. Built for the Auth0 for AI Agents Token Vault Hackathon, it mitigates model drift and prevents privilege escalation during automated infrastructure audits.

---

## Key Features

*   **Zero-Trust Agent Vault:** Securely isolates agent API tokens and restricts execution contexts.
*   **Automated Audit Pipeline:** Evaluates cloud environment policies, identifies configuration drift, and recommends Remediation Action Plans.
*   **Security Guardrails:** Enforces compliance checks dynamically before actions are executed, preventing unauthorized API calls or credential exfiltration.

`Node.js` `Auth0` `Token Vault` `TypeScript` `Azure API` `LLM Orchestration`
